## 💾 Configuration Map

### Environment Variables Flow

```
.env (Docker Host)
  ↓
docker-compose.yml
  ↓
Container Environment
  ↓
.NET Configuration
  ↓
appsettings.json (with Environment override)
```

### Example: Database Connection

```
.env:
MSSQL_CONNECTION_STRING=Server=sql-server,1433;Database=QRDine;User Id=sa;Password=...

docker-compose.yml:
ConnectionStrings__DefaultConnection: "${MSSQL_CONNECTION_STRING}"

.NET Runtime:
Configuration["ConnectionStrings:DefaultConnection"]
  = "Server=sql-server,1433;Database=QRDine;User Id=sa;Password=..."

Entity Framework:
options.UseSqlServer(configuration.GetConnectionString("DefaultConnection"))
```

## 🎯 Production Deployment

Before running `docker-compose up -d`:

1. **Configure .env**
   - [ ] MSSQL_SA_PASSWORD (strong, 12+ chars)
   - [ ] REDIS_PASSWORD (strong, 12+ chars)
   - [ ] JWT_SECRET (unique, strong)
   - [ ] Email credentials (real SMTP)
   - [ ] Cloudinary API keys
   - [ ] PayOS keys
   - [ ] CORS origins (production domains)

2. **SSL Certificates**
   - [ ] Generate real certificates (not self-signed)
   - [ ] Place in certs/server.crt and certs/server.key
   - [ ] Update nginx.conf if using different paths

3. **Database**
   - [ ] Consider SQL Server Standard (not Express)
   - [ ] Plan for backups
   - [ ] Configure maintenance jobs

4. **Redis**
   - [ ] Enable persistence (AOF or RDB)
   - [ ] Plan for cache eviction policy

5. **Monitoring**
   - [ ] Setup log aggregation
   - [ ] Configure alerts
   - [ ] Monitor resource usage

## 📋 Container Services Quick Facts

### API (qrdine-api)

- **Image**: Custom (Dockerfile)
- **Port**: 8080 (internal), 5000 (external)
- **Init Process**: dotnet QRDine.API.dll
- **Health**: GET /health (30s interval)
- **Restart**: unless-stopped

### SQL Server (qrdine-sqlserver)

- **Image**: mcr.microsoft.com/mssql/server:2022-latest
- **Port**: 1433
- **User**: sa (admin)
- **Edition**: Express (10GB max)
- **Volume**: sqlserver_data (/var/opt/mssql)

### Redis (qrdine-redis)

- **Image**: redis:7-alpine
- **Port**: 6379
- **Auth**: Password from .env
- **Persistence**: AOF (appendonly yes)
- **Volume**: redis_data (/data)

### Nginx (qrdine-nginx)

- **Image**: nginx:1.27-alpine
- **Ports**: 80 (HTTP), 443 (HTTPS)
- **Config**: /etc/nginx/nginx.conf (mounted)
- **Certs**: /etc/nginx/certs/ (mounted)
- **Logs**: /var/log/nginx (mounted)
